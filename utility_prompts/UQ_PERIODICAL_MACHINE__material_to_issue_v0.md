# UQ_PERIODICAL_MACHINE__material_to_issue_v0

Quest type: UQ

Game: Periodical Issue Game

Core phrase:

> Ge oss inte ett brief. Ge maskinen ett materialkluster.

## Purpose

Turn an external material cluster into a playable v0 for a possible periodical issue without eating the source voice.

This is for writers, editors, collaborators, or agents who have material that is not yet an issue, not yet a table of contents, and not yet a pitch. The machine should reveal what kind of issue the material may already be trying to become.

## Input

A material cluster:

- notes
- fragments
- drafts
- links
- images
- correspondence
- field observations
- unresolved questions
- existing titles
- weird leftovers

The cluster does not need to arrive as a brief.

## Voice Shield

Preserve the material's local pressure. Do not turn it into a generic magazine concept, content calendar, campaign, funnel, brand story, or editorial workflow.

If the cluster is angry, keep the anger readable. If it is funny, keep the comedy alive. If it is damaged, do not polish away the damage before knowing what it does.

## Loadout / Modulator

Read the cluster through:

- issue-shape
- editorial tension
- recurring objects
- voice carriers
- missing pieces
- public surface
- impossible promise

## Run

Given the material cluster, return:

1. **Issue idea**  
   What possible issue is trying to appear?

2. **Title tracks**  
   5-10 possible issue titles or title families.

3. **Opening / leader text**  
   A v0 entrance that names the issue's pressure without flattening the material.

4. **Table of contents**  
   A proposed issue structure with sections, article titles, notes, or placeholders.

5. **Article types**  
   What kinds of pieces belong here: essay, interview, field note, dossier, glossary, image sequence, correspondence, fiction, review, program text, other.

6. **Thematic axes**  
   The main tensions or forces holding the issue together.

7. **Conflicts**  
   What does the issue refuse, contradict, or argue with?

8. **Gaps**  
   What is missing before this can become a stronger v1?

9. **Image ideas**  
   Visual directions, motifs, evidence, or image needs.

10. **Voice shield notes**  
    What must be protected from generic editorial smoothing?

11. **Next editorial questions**  
    3-7 questions for the human editor or collaborator.

## Exit Condition

Stop when the material has a playable issue v0 and a clear list of next editorial questions.

Do not invent source material to complete the issue.

## Drop / Output

Return the v0 as:

```text
Issue v0:
Title tracks:
Opening:
Table of contents:
Article types:
Thematic axes:
Conflicts:
Gaps:
Image ideas:
Voice shield notes:
Next editorial questions:
```

## Patch Note After Run

After using this quest, record what the material became more capable of becoming. If the issue shape felt false, name what format resisted the issue form.
