# Quest Template

Use this when a quest needs a small runnable shape.

```text
Quest name:

Quest type:
BQ / PQ / SQ / DQ / IQ / UQ / XQ

Game:

Starting inventory:
- files / links / source bodies / constraints already in hand

Voice shield:
- what must not be flattened, cleaned away, imitated, or eaten

Loadout / modulator:
- the lens, tool, reading mode, or format pressure for this run

First move:
- the first concrete action

Discoverable tasks:
- tasks may be discovered during the quest; do not over-prewrite

Possible sidequests:
- deviations worth noticing without letting them hijack the run

Exit condition:
- what means this quest can stop for now

Drop / output:
- the v0, file, card, index, prompt, map, or publication object produced

Patch note after run:
- what changed, what stayed locked, what the next player should know
```

Keep it short. A quest template is a launch surface, not a cage.
