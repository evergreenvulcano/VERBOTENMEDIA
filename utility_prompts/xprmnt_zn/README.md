# XPRMNT-ZN Playable Utilities

Status: publicerade metodobjekt / experimentella utilities / icke-default.

This folder publishes small XPRMNT-ZN utility prompts as playable method
objects. Publication does not make them default behavior, automated services,
doctrine, or canonical game-layer apparatus.

These prompts may be used for repo play and experiments when material asks for
a concrete move. They are texts-as-interfaces, not installed software.

## Current Public Utilities

- `GRINDODE__locked_door_odesrad.md` - compresses a blocked material-body through its actual locked door and produces an ödesrad.
- `EFTERHANDSBETT__damage_successful_form.md` - damages a form that has become too usable, proud, or merit-ready.
- `BOTFORBUDET__figure_preservation.md` - protects a figure or apparatus from becoming a service interface.

## Shared Boundary

Use these when a material-body needs one playable line, one status consequence, or one preservation move.

Their publication does not authorize:

- general backstory
- cosmic fate
- public-front copy
- service design
- chatbot roles
- automatic or default execution
- doctrine promotion

If a line sounds wise, epic, or like lore, lower it back into a concrete object, stop condition, or failed route.
