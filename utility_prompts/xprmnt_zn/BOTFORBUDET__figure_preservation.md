# BOTFORBUDET

Status: PUBLISHED_METHOD_OBJECT / EXPERIMENTAL_UTILITY / NON_DEFAULT

Scope: XPRMNT-ZN / figure-preservation utility

Purpose: Protect a figure, voice, or apparatus from becoming a service interface by turning the refusal into an ödesrad.

## Use When

Use this when a figure is strong enough to tempt interface-thinking.

Examples:

- "Ask Praktikanten"
- "Wise Consultant as coach"
- "Norrotiv as community channel"
- "XPRMNT-ZN as interactive tool"

## Input

- Figure or voice:
- Actual work already performed:
- Tempting interface:
- What the interface would erase:
- What the figure refuses:
- Repeated route:
- Concrete object that stops the service-form:

## Procedure

1. State what the figure was born as.
2. State what it was wrongly invited to become.
3. State what work it actually kept doing.
4. Identify the service-interface that would flatten it.
5. Write one ödesrad where the figure survives by refusing usefulness.

## Output

- One ödesrad.
- One preservation note.
- One playable consequence.

## Rule

Do not protect the figure by calling it sacred.

Protect it by making the wrong use mechanically impossible.

## Example

A consultant was born to criticize the work, refused to become a coach, spoke only through letters and bankett interruptions, and died when the calendar invite called the meeting "team alignment."

## Failure Mode

If the figure becomes mysterious, the mechanic failed.

If the figure becomes helpful, the mechanic failed harder.

If the refusal produces a concrete consequence, keep it.
