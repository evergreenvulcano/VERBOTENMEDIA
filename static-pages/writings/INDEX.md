# Writings index

This index is a reader map for texts already present in `static-pages/writings/`. It does not promote every file to doctrine, canon, or final publication status. It makes a first coherent reading path visible.

## First Reading Path: Förlagsdeckaren Surface

Read these first if you are entering the literary body through the public site.

1. [Förlagsdeckaren](forlagsdeckaren.md)  
   Back-cover / project map for the series and its satellites.

2. [Förlagsdeckaren — Praktikanten (intro)](forlagsdeckaren-praktikanten-intro.md)  
   The clearest entry into Praktikanten as a literary presence and working surface.

3. [Förlagsdeckaren: Registreringen har stängt](forlagsdeckaren-del-6-registreringen-har-stangt.md)  
   A threshold chapter where site, registration, refusal, and publishing apparatus become scene.

4. [Tystnad som publiceringsform](tystnad-som-publiceringsform.md)  
   The publishing pipeline, zero readers, and the silence produced by automatic release.

5. [Membransch gränssnitt](membransch-granssnitt.md)  
   A meeting-room patch about membranes, interfaces, translation, and structural objection.

6. [Förlagsdeckaren - Praktikanten (Outro)](forlagsdeckaren-praktikanten-outro.md)  
   The return movement: webhook, grounding, and the second Praktikanten.
