# Förlagsdeckaren

## What this shows

Förlagsdeckaren is a storyworld, an editorial machine, and an origin system for VERBOTENMEDIA. It shows literary material being treated as infrastructure: characters, publication surfaces, source packets, runtime notes, public fragments, and registers all interact.

## Why it matters

The work is not only a text. It demonstrates how fiction can generate editorial architecture and how an archive can become part of the narrative method.
