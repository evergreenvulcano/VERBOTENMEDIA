# Bankett as Time Engine

Status: canonical doctrine for current public-front build work.
Readiness: provisional.

Note: This doctrine is accepted as a working northstar, but it may need to be rebuilt

## Introduction

A Bankett is not a blog post, progress report, release note, or update.

A Bankett is a ceremonial time marker: a way to give a work period status, closure,
and prospective opening. It is the public time engine of the work.

## What A Bankett Does

### 1. Closure

A Bankett says: this period worked with this material; here is what was learned;
here is what became dangerous; here is what is now possible.

It does not say: we did this, then this, then this; these tasks are done; this
phase is complete.

### 2. Maturation

A Bankett presents maturation as ceremonial fact, not percentage or milestone.

It can say that XPRMNT-ZN is now experimental infrastructure, or that a later phase
has become possible. It must not become "60% done" or "next milestone".

### 3. Prospective Opening

A Bankett ends with possible futures, not instructions.

It says: a future could look like this, this could now be reactivated, this is
possible if conditions change.

It does not say: next we will do this.

### 4. Ceremonial Dignity

The time between Banketter is not dead time. It is work. The Bankett marks it
ceremonially.

## Why It Matters

A progress report asks: how far are we?

A release note asks: what is new?

A Bankett asks: what is possible now?

This makes the Bankett suitable for witnesses. It gives status without creating a
user expectation, product promise, or management interface.

## Form

- Essayistic.
- Swedish by default.
- Long enough to carry ceremony rather than update logic.
- Frontmatter may include title, date, status, relates_to, and phase when the
  Bankett is actually published.
- Ending remains open.

## What A Bankett Does Not Replace

### Not Status Dashboard

A dashboard says where every module is. A Bankett says what became possible in a
period.

### Not Changelog

A changelog says what changed. A Bankett says what was tested and what can now be
borne.

### Not Press Release

A press release announces. A Bankett ceremonially marks.

### Not Release Note

A release note serves users. A Bankett addresses witnesses.
