# Second-brain anti-sentiment

VERBOTEN MEDIA is not a second brain.

It is not an AIOS, life operating system, productivity dashboard, personal knowledge OS, startup knowledge base, or totalizing memory product. The repository is an agent-readable production and publication environment for a literary work and its public surfaces.

## What the repo is

- a static public site shell
- a publishing membrane for public-ready markdown and HTML
- a machine room for source, staging, proposals, operational notes, and development bodies
- a place where status can be made explicit before use
- a collaboration surface for bounded human and Codex work

## What it is not

- a promise to ingest every private source
- a productivity methodology
- a total model of Martin, Verboten Media, or Norrotiv
- a generic AI content pipeline
- a place where public-readable automatically means public-ready

## Diegetic exception

Verboten Media may use brain, skull, skalle, lobe, or second-brain language as literary object, prop, lodging, grotesque interface, or joke when the work calls for it. That does not authorize the repo to describe itself as a second brain.

## Norrotiv posture

Norrotiv should remain anti-second-brain: open to contribution, unfinished in the right way, and available for idea tests without claiming a finished total identity. It receives and routes pressure; it does not become a universal system.
