# Capsule Prompt Battery

This folder contains prompts for converting source material into public-facing knowledge capsules for VERBOTENMEDIA / Textflow Mode.

These prompts are not for rewriting literary prose.

They are for extracting, diagnosing, structuring, and publishing meta-knowledge that arises from source material.

## Source material may include

- essayistic prose with strong authorial voice
- fragments
- notes
- origin documents
- process logs
- agent outputs
- drafts
- comments
- publication reflections
- source-status material

## Core rule

Do not touch the prose.

Do not improve, smooth, paraphrase, imitate, polish, condense, or normalize the authorial voice.

The source material may be read, cited, diagnosed, mapped, and transformed into public capsules, but the capsule must not pretend to be the original literary or essayistic voice.

## Capsule types

This battery supports:

1. Source Intake
2. Härkomstkapsel
3. Pressure Map
4. Publiceringsdiagnos
5. Negativ Poetik
6. AI-statuskapsel
7. Metodfragment
8. Beviskapsel
9. Källsvans
10. Capsule Selection Report

## Publication status

Every output must declare its status:

- authorial source material
- AI-mediated capsule
- editorial diagnosis
- source-status reading
- method fragment
- publication candidate
- internal only
- review required

## Forbidden behavior

The agent must not:

- rewrite the source
- speak as the author
- over-explain the project
- turn the capsule into marketing
- summarize the whole project
- invent provenance
- invent references
- erase uncertainty
- convert fragments into finished essays unless explicitly instructed
