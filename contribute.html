How to approach the body

This is not generic open-source theatre. It is a way to route attention without confusing the bodies.

Fast routing

Use the smallest path: read first, route ideas through Norrotiv, use XPRMNT-ZN for boundary questions, and keep concrete repo edits narrow.

Read first → Idea route → Boundary route → Email route →

If you want to read

Read without obligation. You do not need to become a contributor, collaborator, beta user, or stakeholder. Reading is already a complete relation.

Writings →

If you have an idea

Send it as a bounded proposal: what surface it touches, what it would change, what it should not change, and whether it belongs to Norrotiv, Verboten Media, XPRMNT-ZN, or the machine room.

Email verbotenmedia@protonmail.com →

If you make a PR

A useful PR improves a public surface, clarifies status language, fixes navigation, adds a reviewed static page, or documents a boundary. It should not bulkimport Drive, normalize raw source dumps, rewrite literary prose, or promote staging material as canon.

Repo collaboration contract →

Which body are you contacting?

Verboten Media: publishing, literary work, entertainment apparatus, availability, public inquiries.

Norrotiv: idea tests, collaborator pressure, PRs, external proposals, DragonKeep-like adjacency, unfinished but bounded interaction.

Martin: authorial questions, interviews, literary context, and the distinction between method and work.
