# Inbox Writings

Files placed in this folder are converted candidates, not yet canonical published writings.

Promotion target:

- `static-pages/writings/`

Before promotion, review each file for:

- title
- summary
- type
- date
- naming
- Markdown Publishing Standard v1 compliance

## Conversion integrity

Conversion itself does not authorize creative-text editing.

Allowed during conversion:

- adding flat frontmatter required by the repository contract
- assigning compliant file names and metadata fields
- preserving or mapping existing structural headings
- converting only container-level formatting into supported markdown structures where the source wording is unchanged
- flagging unsupported structures or incompatibilities
- staging the result here before promotion

Not allowed during conversion unless explicitly requested:

- changing words, sentences, or paragraphs
- changing tone or rhetorical character
- readability edits
- punctuation or style normalization
- paraphrase
- semantic compression
- collapsing repetitions
- editorial cleanup disguised as conversion

If compliance would require altering the creative text, flag the incompatibility instead of rewriting the file.

Do not treat files in `inbox/writings/` as published assets until they have been reviewed and intentionally promoted.
