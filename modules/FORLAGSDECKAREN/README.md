# FORLAGSDECKAREN

Developmental runtime module for Förlagsdeckaren.

This module is non-canon and non-public. It is not part of the markdown publishing surface, must not be wired into `writings.html`, and carries no publishing authority.

## Purpose

`FORLAGSDECKAREN` holds source-informed runtime infrastructure for future chapter and scene composition, including Praktikanten-as-literary-engine and multi-character board room formats.

It is conceptually adjacent to `_DUJAG` as a support runtime package, but it is independent from `_DUJAG`. DUJAG may be used as architectural precedent only; the two modules must not be merged.

## Boundary

Folder existence, organization, or legibility does not constitute canonization. The material here remains developmental until a separate explicit editorial decision promotes a derivative output.

Promotion to public writing requires a separate human editorial decision and a deliberate move into the public writing surface. No file in this module should be treated as publication copy by default.

## Anti-expansion clause

This module must not grow by self-justification. Add files only when a specific chapter/runtime need requires them. More scaffolding is not proof of better runtime discipline.
