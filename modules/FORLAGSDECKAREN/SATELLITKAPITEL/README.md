# Satellitkapitel

This folder contains satellite-chapter artefacts connected to Förlagsdeckaren.

These texts are internally significant creative artefacts and continuing evidence of the project's editorial capacity. They are preserved here as module-grade material, not as automatically promoted public canon.

Files in this folder are not wired into public site publication surfaces unless separately promoted through the appropriate repo process.

## Local conversion standard

All future conversions into this folder must follow preservation-first conversion.

- Convert source files into clean Markdown.
- Preserve source wording.
- Do not rewrite, smooth, paraphrase, condense, censor, modernize, or tonally improve.
- Perform format-only normalization unless a change is required to repair conversion artifacts.
- Preserve literary instability, abrasive language, surreal syntax, fragmentary structure, deliberate irregularity, and strange style when intentional.
- Use repo-normalized filenames. If no stronger local convention exists, use lowercase kebab-case ASCII slugs.
- Preserve full Swedish titles and characters inside Markdown content.
- Use flat frontmatter only if compatible with repo standards.
- Do not use nested YAML, arrays, multiline frontmatter values, or unsupported frontmatter structures.
- Do not wire files into public publication indexes.
- Keep public promotion as a separate explicit action.
- If embedded images/assets exist, place them in a local assets folder and update links.
- If conversion fails for a file, do not abort the whole batch; report the file-specific failure.

## Promotion note

Promotion from this folder into public/canonical publication surfaces must be handled as a separate explicit task. Promotion must not be inferred from conversion.
