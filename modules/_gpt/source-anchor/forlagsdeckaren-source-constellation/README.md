# Forlagsdeckaren Source Constellation

This folder preserves source anchors that were previously held under a temporary chapter-number packet structure.

The active orientation is now functional rather than sequential. The retired chapter-number structure remains historical provenance only, documented in `legacy-provenance.md`.

## Rule

Use these packets by source function, not by old reading order.

The material here is source/provenance. It is not public content, not canon, not a publishing manifest, and not a reason to mutate outward-facing files. Binary sources are intentionally kept as binary sources until a separate conversion task is explicitly requested.

## Packets

- `threshold-and-public-surface/`: strategic letter, website source, ideation material, and workshop-shell threshold logic.
- `heading-extraction-and-deployment/`: heading doctrine, aphorism extraction workflow, schema, deployment skeleton, and reframing substrate.
- `infrastructure-migration-and-shadow-copy/`: pre-migration artefact body, infrastructure theory, ritual prompt, node, style, and shadow-copy architecture.
- `archive-object-cartography-and-signals/`: withheld novel-object references, mindmap and screenshot evidence, cult edition logic, black-box report, and signal architecture.
- `abdication-pipeline-and-surplus/`: exit interview source, threshold draft, content-vacuum fax, campaign timing, surplus artefact, analytics, return signal, pipeline, dashboard, silent synapses, and reflex logic.
- `grounding-tankefigur-and-bifigure/`: grounding register, Tankefigur method documents, companion Praktikanten body, post-it cartography, page trigger, and bifigure reference.

## Handling

- Preserve source wording.
- Do not convert docx/odt files in this pass.
- Do not treat these packets as public-ready writing.
- Do not add these sources to `source-manifest.md` without a separate source-anchor review.
- Route historical path questions to `legacy-provenance.md`.
