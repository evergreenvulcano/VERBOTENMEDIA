# Grounding Tankefigur and Bifigure

This packet preserves sources where grounding becomes method, method becomes metadata, and the grounder splits across parallel bodies without collapsing character invariance.

## Included Sources

- `twenty-card-grounding-register-reference.md`: reference note for the existing repo-local grounding register.
- `Metod_Tankefigur som metadata.docx`: method source with keynote, ritual sequence, submission directives, micro-interactions, and metadata rhetoric.
- `Metod_ Tankefigur som ontologisk design av totalprojekt.docx`: companion method source for broader ontological-design framing.
- `Den andra Praktikanten.docx`: companion character body preserving parallel-Praktikanten architecture.
- `post-it-cartography-reference.md`: physical installation and spatial grounding note.
- `page-203-reference.md`: page-trigger note for the parallel-body recognition event.
- `bifigur-reference.md`: missing-card / gravitational-center note.

## Use

Use this packet when orienting work around grounding, Tankefigur method, metadata as ritual surface, physical cartography, bifigure logic, and character invariance across agent/body variation.

Do not treat grounding as closure. The packet preserves grounding as a method event that splits the grounder while keeping the character field intact.
