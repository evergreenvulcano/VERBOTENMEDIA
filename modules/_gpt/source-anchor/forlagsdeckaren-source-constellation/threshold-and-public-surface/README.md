# Threshold and Public Surface

This packet preserves sources where Verboten Media becomes a visible threshold: a strategic letter, a functioning website artefact, ideation material, and a workshop shell whose emptiness is meaningful.

## Included Sources

- `fragment_stratcom.docx`: strategic letter source for closed-registration language, Academy desire-through-denial logic, and the `Kara xyz` publisher address.
- `index-html-source.txt`: functioning site source with black/white serif surface, restrained accent color, section architecture, low-frequency contact logic, and refusal-as-interface.
- `hemsida_ideboll.docx`: broader institutional ideation source behind the public surface, with vocabulary around Flempo, lectures, workshops, artefacts, forums, and rotating taglines.
- `pre-rationell-modularitet-i-spegelproxess.docx`: workshop-shell source retained as a meaningful empty/incomplete instructional surface.
- `pre-rationell-modularitet-i-spegelproxess-reference.md`: local reference note for the workshop shell.

## Use

Use this packet when orienting work around access denial, public surface as narrative method, website-as-threshold, and the gap between brainstorm, built page, and institutional self-description.

Do not reduce this packet to ordinary website material. Its value is the shared pressure between strategic address, built interface, ideation surface, and inaccessible workshop.
