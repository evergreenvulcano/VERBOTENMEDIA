## Capsule

A literary character can operate a website when the page does not merely present the character, but lets the character administer passage.

In this artifact, the operative figure is not treated as biography, mascot, or lore. The figure becomes a web-surface logic: registration closes, links withhold themselves, the archive reveals by delay, contact becomes movement, and refusal becomes a form of meaning.

The discovery is not that fiction can be placed on a website. The discovery is that a website can become a character-governed threshold.

## Function

VERBOTOM ACADEMY tests whether Praktikanten / threshold logic can move from literary scene into interface behavior without becoming ordinary character branding.

The useful result is the form:

- character as interface administrator
- refusal as interaction logic
- archive as delayed organ
- signup as closed ritual
- contact as movement trace
- surface path vs residue path as epistemology
