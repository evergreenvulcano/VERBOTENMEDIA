# index

Stable map layer for `modules/_gpt/`.

Use this folder for durable orientation surfaces:
- source manifests
- node registries
- corpus maps
- adjacency maps
- target-path inventories
- source-to-output pointers

This folder should not become a diary.
It should remain relatively calm and structurally useful.

## Good contents
- `source-manifest.md`
- `node-registry.md`
- `adjacency-map.md`
- `target-paths.md`

## Bad contents
- raw brainstorming
- long daily logs
- duplicate copies of source text
- half-finished prose with no structural purpose

If something is time-bound or session-bound, it probably belongs in `../runs/`, not here.
