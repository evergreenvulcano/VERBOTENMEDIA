# Tankefigur

`tankefigur` is a developmental method/runtime module. It preserves the raw `TOTAL_EVENT0` source and derives operational surfaces from the Tankefigur / Thought-Figure method.

This module is non-public. It is not public copy, not canon by folder existence, and not part of the markdown publishing surface. It must not be wired into `writings.html` or promoted into `static-pages/writings/` without an explicit human editorial decision.

The module is conceptually near `modules/klangbotten_snippets.md` and `modules/executive_summary.md`, but it does not replace them. Existing references are treated as partial orientation, not complete representation.

The module exists to make underused method material usable by future agents, runtimes, writing systems, and project reviews.

Normal folder naming is intentional: Tankefigur is a named method surface, not hidden implementation machinery.
