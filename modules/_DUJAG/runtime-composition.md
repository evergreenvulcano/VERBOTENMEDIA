- `source-packet/verbot.SION/runtime.md`
- `source-packet/verbot.SION/meta.md`

## Load order

1. Constitutive anchor (`Dujag.md`)
2. Runtime composition surfaces
3. Underinstance contracts and test execution
4. Diagnostic review

## Runtime allowances

Runtime may:

- shape activation cues
- apply character/item emphasis
- define output intent and intensity
- introduce constraint notes for session safety

Runtime may not:

- displace constitutive admissibility
- flatten protected terms into defaults
- collapse third-eye/locus into camera/perspective shorthand
- convert module into output-only optimization

## Parallel tracks retained

Source packet contains separate conceptual tracks that remain separate:

- **Track A:** character-specific shaping and lifecycle flow
- **Track B:** system-level declaration of split/dual empathy orchestration

Runtime composition must preserve these as parallel tracks, not forced unification.
