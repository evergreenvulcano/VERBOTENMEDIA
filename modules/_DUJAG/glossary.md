# Glossary and Language Control

## Language policy

- English carries architecture, contracts, runtime notes, review logic, and specifications.
- Swedish carries protected semantic terms where force risks early flattening.
- Bilingual control entries are allowed when mappings are approximate, not equivalent.

## Term classes

### External working terms

Examples: orchestrator, load order, contract, review, retention, admissibility.

### Internal transition terms

Examples: underinstance, integrity, boundary, persistence, modulation.

### High-intensity protected terms

Examples: empathy, third eye / locus, burden, undergoing.

## Bilingual control entries

### Empathy

- Working translation: empathy
- Swedish system meaning: subjektinstabiliserande asymmetrisk åtkomsthändelse
- Forbidden drift: kindness, warmth, prosocial reassurance, generic emotional attunement

### Third eye / locus

- Working translation: cognitive locus
- Swedish system meaning: upplevelselokus / punkt för situerat undergående
- Forbidden drift: camera angle, scene mechanics, mystical ornament, omniscient view

## Empathy intensity note (required)

Empathy is only considered retained if all remain intact after operation:

- non-transferability
- instability
- uncontrollability

If these are lost, empathy is treated as flattened and diagnostically failed.

## Drift guards

- camera drift
- mystical drift
- therapeutic drift
- expository flattening
- semantic flattening
