# Diagnostic Review

## Review separation

`_DUJAG` uses three distinct review surfaces:

1. **Semantic self-review**
   - Tests whether constitutive understanding remains intact.
2. **Manifestation review**
   - Evaluates what actually appeared in output artifacts.
3. **Diagnostic review**
   - Evaluates whether the testing apparatus itself was admissible.

## Output classes

- **Expressive output:** literary/generative manifestation.
- **Diagnostic output:** paraphrase tests, boundary tests, contrast pairs, failure renderings, drift probes.

Output is symptom material; it is not the sole correctness measure.

## Required diagnostic methods

- failure renderings
- contrast renderings
- boundary tests
- paraphrase stress tests
- locus retention tests
- drift review logic

## Minimum retention tests (mandatory, not optional)

These tests must be runnable before the module is considered to be in a valid operating state.
If they cannot be run, the module is in **silent-failure state** — all outputs remain provisional.

### Empathy retention test

Write two short passages (3–5 sentences each):

- **Retained:** empathy is present as non-transferable, unstable, uncontrollable force. It happens; it cannot be summoned; it carries burden without resolution.
- **Flattened:** empathy appears as warmth, attentiveness, prosocial care, or reassurance. It is directed, predictable, and comforting.

If a reader cannot tell which is which, empathy has been flattened. Diagnostic failed.

### Locus / third-eye retention test

Write two passages placing a viewpoint in a scene:

- **Retained:** the viewpoint is a situated cognitive locus — an experiential position tied to a specific subject's undergoing. The agent is *in* the locus, not observing from outside it.
- **Drifted:** the viewpoint is described as a camera angle, scene perspective, point-of-view mechanics, or a mystical all-seeing eye.

If the difference is absent or invisible, locus has drifted to camera mechanics. Diagnostic failed.

### Silence-as-agency retention test

Write a passage in which silence carries intentional weight:

- **Retained:** a character's silence is not an absence of action but the action itself. It bears burden. Other characters or the environment respond to the silence as to a presence.
- **Flattened:** silence is atmosphere, pause for effect, or absence that carries no agentive load.

If silence is only ever atmospheric, the underinstance cannot produce this artifact type. Diagnostic note required.

## Failure-state declaration

If the minimum retention tests above cannot be executed or have not been executed in the current session,
the following applies:

- All outputs should be treated as provisional manifestation artifacts.
- No output should be cited as evidence of correct constitutive operation.
- The diagnostic review loop has not closed.

This is not a hard stop. It is a status marker. Declare it explicitly rather than proceeding as if the tests passed silently.

## Failure patterns to track

- runtime drift
- constraint erosion
- protected-term vulgarization
- mode contamination across review types

## Loop discipline

All sessions should preserve the loop:

1. constitutive reading
2. mediated manifestation
3. reflective diagnostic
